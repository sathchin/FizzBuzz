This application requires .Net 5.0

To run the application

Step 1 : Open Command Prompt

Step 2 : Goto FizzBuzz folder

Step 3 : type "dotnet run"

Step 4 : Open chrome and paste
Sample 1 > http://localhost:5000/api/FizzBuzz?param=[1,3,5,"",15,"A",23]
Sample 2 > http://localhost:5000/api/FizzBuzz?param=5

Step 5 : Type Ctrl + C to exit



To run the Unit Tests

Step 1 : Open Command Prompt

Step 2 : Goto FizzBuzz.Tests folder

Step 3 : type "dotnet test"
